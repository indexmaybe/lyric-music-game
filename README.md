## **# My Lyric Arcade Game 🎵**



Hey there! This is my Python project for the Horizons coding event. I wanted to build a simple game that gives you lyrics as text for a song and you have to type the missing word.



the game is built using python and Tkinter with a little help with Ai for explaining confusing errors



###### **Stuff It Can Do**



* Skins / Themes You can open the dropdown menu in the top-right corner and change the game's colors

&#x20;  (Arcade Purple, Matrix Green, or Cyberpunk Red).



* High Score Saver It automatically creates a local file called`arcade\\\_save.json to save your highest score and how many times you've played, even if you close the app.



* Live Timer There is a 30-second countdown clock built into the screen. If you don't answer in time, its game over



* Widescreen Mode I added a button that lets you pop the window open into a wider layout if you want more room.



* Keyboard Support Instead of clicking the submit button you can just type your answer and hit the "Enter" key



###### **How I Made It:**

Language: Python 3

Library: Tkinter (for the window and buttons)

Time Spent: Coded entirely by hand in VS Code, tracked via Hackatime.



##### ***How to Play:***

##### ***Just run the `main.py` script on your computer:***
